Projetos

Fred Games

Jogo 2D desenvolvido em Python onde você controla Fred Flintstone em uma corrida espacial repleta de meteoros. Com controles simples e visuais estilo desenho animado, o jogo desafia reflexos rápidos e proporciona diversão com uma trilha sonora pré-histórica futurista.

Jogos Mortais

Protótipo de jogo em Python que explora lógica condicional e fluxo de decisões interativas. Um projeto acadêmico que demonstra habilidades em programação e estruturação de código para criar experiências simples de jogo em console.

Biblioteca Virtual

Sistema em Java para gerenciamento de livros, incluindo livros físicos e digitais. Permite inserção, consulta e organização de obras em uma biblioteca virtual, demonstrando habilidades em orientação a objetos e programação estruturada.

Tratamento de Dados

Notebook em Python desenvolvido no Google Colab para análise de dados, experimentos ou aprendizado de máquina. Demonstra capacidade de manipulação de dados, visualização e execução de projetos interativos no ambiente do Colab.

Estrutura de Loja

Trabalho acadêmico em Python que implementa um mini sistema de estoque e vendas no terminal, aplicando conceitos de Estruturas de Dados clássicas como listas, pilhas e filas, aliados ao paradigma orientado a objetos (POO).

Cirífood

Projeto feito no Figma para um aplicativo de delivery, com interfaces de baixa fidelidade. O objetivo foi estruturar o fluxo principal do usuário (cadastro, navegação, escolha de produtos e finalização do pedido), priorizando usabilidade, simplicidade e clareza visual.
